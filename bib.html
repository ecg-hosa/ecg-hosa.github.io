About the ReDesign

The original design for the ECG HOSA website was highly dependent on HTML5 UP's Phantom design. This redesign is intended to become more original, independent, and unique, but still maintain the same level of functionality that the original had.

To see the original website, visit archived.


via FY Springfield on GIPHY

Design Process

The main color, #A51B1F, is a lighter version of #841619, which is the official HOSA red color. The gradient was found on CronColor using #841619. All other variations of red found on this page are the same as #A51B1F, but in different opacities, at 50% (80) and 90% (E6).

The fonts used on the site are Roboto Slab (for the main text), Bebas Neue (for most headings), and Oswald (for the navigation menu and some headings). They are all free under Google Fonts.
